# 🍞 Bakery Website

A responsive bakery landing page built using **HTML5** and **CSS3**.\
This project represents a modern bakery website with a clean design,
multiple sections, and a responsive layout.

------------------------------------------------------------------------

## 🔗 Live Demo

 https://abdelhamid1235.github.io/bakery

------------------------------------------------------------------------

## 📌 Features

-   Responsive design
-   Clean and modern UI
-   Landing hero section
-   About section
-   Our Location section
-   Baking gallery
-   Contact section with Google Map
-   Footer with social media icons

------------------------------------------------------------------------

## 🛠️ Built With

-   **HTML5**
-   **CSS3**
-   **Responsive Design (Media Queries)**

-------------------------------------------------------------------------

## 📷 Website Sections

The website contains the following sections:

-   Header / Navigation Bar
-   Landing Section
-   Who We Are
-   Our Location
-   Our Baking
-   Contact Information
-   Footer

------------------------------------------------------------------------

## 📱 Responsive Design

The layout adapts to different screen sizes including:

-   Desktop
-   Tablet
-   Mobile

This is achieved using **CSS media queries**.
